## Work Done
To Install and Supply a one-off BCSP Incline Lift

Functional and Flexible
The BC lift is designed for straight staircases for both indoor and outdoor installation and can be fitted directly to a wall or supported on stanchions.

When the lift is not in use, it can be folded and parked and measures only 350 mm. from the wall, allowing clear access to the staircase. the user can call the lift, even when folded.
